# cloud-pipelines

Affiliate site for CloudPipelines (infrastructure/FinOps audit lead gen). Built for **public** GitHub Pages at [cloudpipelines.com](https://cloudpipelines.com).

- **No secrets**: No API keys, tokens, or env files. Tally form ID and affiliate URLs are public by design.
- **Disclosures**: Affiliate/commission disclosures appear on index, comparison, success, and cloudways pages.
- **Claims**: Copy uses supportable phrasing (e.g. “around 30%” cloud waste per industry reports; Cloudflare on Cloudways described as add-on, not included standard).
